# よけろ

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
## ゲームの概要

* 線と球をよける
* ジャンプ可。
* ライフ3つがなくなったらゲームオーバー。
* 参考サイト→https://qiita.com/KMim/items/24a1d69833bc335779a4

ジャンプや上下左右にプレイヤーである緑の玉を動かして、線と球を避ける。

* 線とドットの親クラス
* 垂直の線に関するクラス
* 水平の線に関するクラス
* ドットに関するクラス

##　追加機能BGM
* ゲーム開始前にオープニングBGMが再生される
* ゲームがスタートするとBGMが再生される
* Tabキーを押すとBGMが切り替わる
* ゲームオーバーするとゲームオーバーのBGMが流れる
